1. Project Overview & Objectives

The Student Information System is a console-based Java application designed to store and manage student records efficiently using Object-Oriented Programming principles.

Objectives:

Practice Java OOP

Implement input validation

Build menu-driven applications

Display formatted output

2. Setup & Installation Instructions

Install Java JDK (8 or later)

Clone the GitHub repository

Open the project in IntelliJ / Eclipse / VS Code

Run Main.java
